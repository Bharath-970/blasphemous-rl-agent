# Blasphemous 2 – Reinforcement Learning Agent

This project trains an AI agent to control a character in the game Blasphemous 2
using reinforcement learning and real-time screen capture.

## How it works
- The agent observes the game via screen pixels
- Uses a CNN-based PPO policy
- Controls the game using keyboard input
- No game API or internal memory access

## Tech Stack
- Python
- Stable-Baselines3 (PPO)
- Gymnasium
- OpenCV
- PyAutoGUI
